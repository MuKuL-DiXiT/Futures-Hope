<h1>Introduction</h1>

Futures-Hope is a learning-focused project that shows how a normal person can create a complete web application using JavaScript on both frontend and backend.

<h2>It includes:</h2>
-A React UI created with Vite
-A backend server created using Express
-Simple API routes
-Basic folder structure that beginners can easily follow

<h1>Tech Stack</h1>
<h2>Frontend</h2>

React, Vite, JSX, Tailwind, lucideReact

<h2>Backend</h2

Node.js, Express, nodemon (for development), socket.io, MongoDB

<h2>Tools</h2>

npm (package manager), Git (version control)

<h1>How Anyone Can Create This Project</h1>

This is the simplest explanation for a beginner. <br>

<h2>--clone the project</h2>

1️⃣ Create the project folder<br>
-- mkdir Futures-Hope<br>
-- cd Futures-Hope<br>
-- cd frontend<br>
-- npm i<br>
-- npm run dev<br>

2️⃣ Create the backend (Express)<br>
-- mkdir backend<br>
-- cd backend<br>
-- npm init i<br>
-- create your own cluster and google client id for database connectivity and google auth testing and set up the env for the same<br>
-- npm start<br>
